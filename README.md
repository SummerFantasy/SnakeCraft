# SnakeCraft
## 游戏玩法
### 单人模式:
* 鼠标控制移动方向
* 吃彩色小球可以增加长度和增大体型
* 当长度大于300时按住鼠标左键可以加速移动
* 超过边界或者撞上其他贪吃蛇将死亡，可复活
* 死亡会损失一定的长度和体型，掉落一定数量的能量球，作用等同彩色小球
* 本场中长度大于1500的贪吃蛇将获胜结束游戏
* 按下ESC将暂停游戏，再次按下恢复游戏
